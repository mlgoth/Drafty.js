Drafty.js supports multiple generations of drafts. In the demonstration setup, a new draft generation is created once per minute. You can preserve as many generations as needed, though this demo is limited to 25 to avoid spamming the demo database.
Drafty.js supports JavaScript callback functions while saving drafts. This can be used to display a simple "Saving" text to the user. In this demonstration all autosaved input fields flash green when saving starts and returns to normal color when the save is complete.

The demonstration page automatically saves changed input fields every few seconds by default. The user can freely change the save interval or disable autosaving. With autosave disabled, user must use the Save-button to save drafts manually.

The web page can also save drafts by calling the Drafty.save_draft() method as needed.

The text you input on this page is automatically saved to a MySQL database. Once a draft is saved, the text is remembered for your future visits.

You can close the browser tab, reload the page, restart the browser or experience a power outage, but when you come back here, your latest draft text will be restored automatically.
